Política de Privacidade

A extensão SpreadBot Automation não coleta, armazena, processa ou compartilha dados pessoais dos usuários.

Dados acessados

A extensão acessa apenas o conteúdo das páginas web abertas pelo usuário com o único objetivo de automatizar interações locais, como cliques e preenchimento de campos, conforme ações explicitamente configuradas pelo próprio usuário.

Armazenamento e compartilhamento

Nenhuma informação é enviada para servidores externos. Todos os dados permanecem no navegador do usuário.

Código remoto

A extensão não utiliza código remoto, scripts externos ou bibliotecas carregadas de servidores de terceiros.
